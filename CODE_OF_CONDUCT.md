# Code of Conduct

## Expected Behavior

- Be direct, respectful, and evidence-based.
- Keep reviews focused on the work and its operational impact.
- Document decisions so future maintainers can understand context.
- Raise security and reliability concerns early.

## Unacceptable Behavior

- Harassment or personal attacks
- Publishing secrets or sensitive data
- Bypassing review or security controls without documented approval
- Misrepresenting validation status

## Enforcement

Repository owners are responsible for moderation and enforcement. Serious issues
should be escalated through the organization's normal governance path.
