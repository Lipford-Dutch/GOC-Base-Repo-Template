# GOC-Base-Repo-Template

This documentation defines the reusable kickoff baseline for new Lipford-Dutch
repositories.

Use it when starting a project that needs:

- clear ownership and repository governance
- secure GitHub Actions defaults
- documentation and Pages publishing
- issue and pull request triage standards
- release and operational readiness guidance

## First Action

Start with the [kickoff checklist](kickoff-checklist.md). It turns a blank
repository into a reviewed, documented, and automation-ready project.

## Brand Asset

Use `docs/assets/social-preview.png` as the base GitHub Social Preview image
for this repository or as a starter asset for child repositories.

## Design Intent

The template is meant to be copied forward, not admired. Delete what does not
apply, fill in ownership, and keep all project-specific decisions in the child
repository.
